# CraftFlow 🎨
### Multi-Tenant SaaS Platform for Home-Based Businesses

CraftFlow is a production-grade platform built for handmade craft sellers, bakers, candle makers, jewelry creators, and custom gift businesses — allowing multiple businesses to operate independently within a single system.

---

## Tech Stack

**Backend**
- Java 21 + Spring Boot 3.2
- Spring Security + JWT Authentication
- PostgreSQL + Spring Data JPA + Flyway
- Docker

**Frontend**
- React 18 + TypeScript
- Redux Toolkit + React Query
- Tailwind CSS
- Vite

---

## Features

- **Multi-Tenancy** — Complete data isolation per business workspace
- **Order Management** — 7-stage workflow state machine (Received → Delivered)
- **Kanban Board** — Drag-and-drop order tracking with workflow validation
- **Production Planning** — Auto-prioritizes orders by deadline, detects delays
- **Inventory Management** — Raw materials, stock in/out, low-stock alerts
- **Customer CRM** — Profiles, order history, lifetime value tracking
- **Product Catalog** — Categories, pricing, stock tracking
- **Role-Based Access** — Owner, Staff, Customer roles
- **Notifications** — Real-time alerts for orders and inventory
